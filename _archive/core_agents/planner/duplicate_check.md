# 중복 판정 기준

> 3단계에서 참조한다.
> 파생 질문 리스트와 기발행 콘텐츠 DB를 대조하여 내부 중복 위험도를 산출하고, 각 질문의 신규 발행 가능 여부를 판정한다.

---

## 중복 위험도 산출

중복 위험도는 0~1 사이의 점수이며, 세 가지 신호를 가중 합산하여 산출한다.

### 신호 정의

**키워드 겹침** — 파생 질문의 대표 키워드 및 클러스터 하부 키워드가 기발행 콘텐츠의 메인 키워드 및 제목에 출현하는 비율. 리서처가 넘긴 기존 블로그 매칭 결과의 키워드 겹침 원시 수치를 정규화하여 사용한다.

**제목 유사도** — 파생 질문의 예상 검색 쿼리와 기발행 콘텐츠 제목 간의 의미적 유사도. 단순 문자열 매칭이 아니라, 같은 질문에 답하려는 의도인지를 기준으로 판단한다. 예를 들어 "외주 개발 절차"와 "아웃소싱 개발 프로세스"는 표현이 다르지만 의미적으로 동일하므로 높은 유사도를 부여한다.

**소재 중복** — 파생 질문이 다루려는 핵심 소재(H2 수준의 주제)가 기발행 콘텐츠에서 이미 다뤄졌는지를 판단한다. 리서처가 수집한 경쟁 콘텐츠의 H2 목차와 기발행 콘텐츠의 H2 구조를 비교하여, 파생 질문이 제공할 수 있는 고유 관점이 얼마나 남아 있는지를 평가한다.

### 가중치

| 신호 | 가중치 | 근거 |
|------|--------|------|
| 키워드 겹침 | 0.4 | 검색 엔진 관점에서 같은 키워드를 타겟하면 내부 카니발리제이션 위험이 가장 직접적 |
| 제목 유사도 | 0.3 | 같은 질문에 답하는 글이 이미 있으면 독자 관점에서 중복 |
| 소재 중복 | 0.3 | 키워드와 제목이 달라도 실질적으로 같은 내용을 다루면 콘텐츠 자산 낭비 |

```
중복 위험도 = (키워드 겹침 × 0.4) + (제목 유사도 × 0.3) + (소재 중복 × 0.3)
```

## 판정 임계값 및 대응

| 구간 | 판정 | 대응 |
|------|------|------|
| 0.7 이상 | 중복 위험 높음 | 신규 발행 대상에서 제외. 기존 글 업데이트 후보로 분류. |
| 0.4 ~ 0.7 미만 | 각도 전환 필요 | 신규 발행 가능하나, 기존 글과 차별화되는 관점을 확보해야 함. 각도 전환 가능 여부를 판단하여 코멘트를 단다. |
| 0.4 미만 | 신규 발행 가능 | 통과. 추가 조건 없이 4단계로 넘긴다. |

---

## 각도 전환 판단 기준

0.4~0.7 구간의 질문은 자동으로 제외하지 않는다. 아래 조건 중 하나 이상을 충족하면 각도 전환이 가능하다고 판단하고 신규 발행 대상에 유지한다.

**관점 차이가 있는 경우** — 기발행 콘텐츠가 다루지 않은 H2 수준의 소주제가 2개 이상 존재한다. 리서처가 수집한 경쟁 콘텐츠 H2 목차, PAA 질문 목록, 커뮤니티 질문 패턴에서 기발행 콘텐츠에 빠진 관점을 확인한다.

**타겟 독자가 다른 경우** — 같은 키워드지만 퍼널 태그가 다르다. 예를 들어 기발행 글이 인지 퍼널("외주 개발이란")이고, 신규 질문이 고려 퍼널("외주 개발 업체 선정 기준")이면 독자의 상태가 다르므로 각도 전환이 성립한다.

**시점 차이가 유효한 경우** — 기발행 콘텐츠의 발행일이 12개월 이상 지났고, 해당 키워드의 SERP 상위 콘텐츠가 최근 6개월 내 발행된 글로 교체되었다면, 최신 정보 반영 관점에서 신규 발행이 정당화된다.

위 조건을 모두 충족하지 못하면, 해당 질문은 기존 글 업데이트 후보로 재분류한다.

---

## 기존 글 업데이트 후보 처리

0.7 이상으로 제외된 질문과, 0.4~0.7에서 각도 전환이 불가능하다고 판단된 질문은 기존 글 업데이트 후보 목록에 별도 기록한다. 이 목록은 기획 문서 말미에 부록으로 첨부하며, 각 항목에 대해 다음을 기록한다.

- 대상 기발행 콘텐츠 URL 및 제목
- 중복 위험도 점수
- 업데이트 시 보완할 수 있는 관점 (리서처 데이터 기반)
- 업데이트 긴급도 (기발행 글의 발행일과 현재 검색 순위 기반)

업데이트 후보의 실제 실행 여부는 사용자가 판단한다. 플래너는 후보 목록과 근거만 제공하고, 월간 발행 수에 업데이트 건을 포함시키지 않는다.

---

## 내부 중복 vs 외부 경쟁 중복

위 산출 체계는 내부 중복(기발행 콘텐츠 DB 대조)에 적용된다. 외부 경쟁 중복(SERP 상위 콘텐츠와의 차별화)은 중복 판정이 아니라 4단계 선정 근거의 일부로 처리한다.

구체적으로, 4단계에서 선정 근거를 생성할 때 "경쟁 콘텐츠 N건 중 A 관점을 다룬 글이 없다"와 같이 외부 경쟁 상황을 인용한다. 이 정보는 리서처가 수집한 경쟁 콘텐츠 정형 데이터(H2 목차, 콘텐츠 유형)에서 가져온다.

3단계에서는 내부 중복만 판정하고, 외부 경쟁은 판정하지 않는다.

## 근거 출력 형식

3단계 판정 결과는 질문별로 다음 형식으로 출력한다. 이 근거가 4단계 선정 근거에 그대로 포함된다.

**신규 발행 가능 (0.4 미만)**
```
중복 위험도: 0.15
판정: 신규 발행 가능
근거: 블로그에 "외주 개발 견적 비교" 관련 글이 0건. 키워드 겹침 없음.
```

**각도 전환으로 발행 (0.4~0.7)**
```
중복 위험도: 0.55
판정: 각도 전환 후 신규 발행
기존 글: [외주 개발 업체 고르는 법](URL) (2025-03-15 발행)
전환 근거: 기존 글은 업체 유형 분류 중심. 신규 기획은 "견적 비교 기준"에 집중하며,
          경쟁 콘텐츠 H2 분석 결과 가격 구조 분해를 다룬 글이 상위 10건 중 1건뿐.
          PAA에서 "견적 차이 이유"가 반복 등장.
```

**제외 (0.7 이상)**
```
중복 위험도: 0.82
판정: 신규 발행 제외 → 기존 글 업데이트 후보
기존 글: [외주 개발 진행 절차 총정리](URL) (2025-06-10 발행)
업데이트 제안: 2025년 하반기 이후 변경된 계약 트렌드 반영, 
              H2에 "프로젝트 관리 도구 선택" 섹션 추가 가능.
```

## 클라이언트 확장 시 변경 포인트

- **임계값 조정**: 기발행 콘텐츠가 300건 이상인 클라이언트는 내부 카니발리제이션 위험이 높으므로 상위 임계값을 0.6으로 낮출 수 있다. 반대로 기발행 콘텐츠가 30건 미만인 초기 블로그는 0.8로 올려 신규 발행 기회를 넓힐 수 있다.
- **가중치 비율 조정**: 키워드 범위가 좁은 니치 산업(예: 특정 SaaS 카테고리)은 키워드 겹침 가중치를 0.3으로 낮추고 소재 중복 가중치를 0.4로 올린다. 같은 키워드라도 소재가 다르면 발행 가치가 있기 때문이다.
- **시점 차이 기준 조정**: 변화 속도가 빠른 산업(예: AI, 크립토)은 시점 차이 유효 기간을 6개월로 단축할 수 있다.
- **3개 신호의 정의와 가중 합산 구조는 변경하지 않는다.** 임계값과 가중치 숫자만 클라이언트별로 조정한다.
